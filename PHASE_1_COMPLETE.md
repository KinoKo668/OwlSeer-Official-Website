### **3. 优化了性能**

**懒加载：**
```typescript
const HomeMobile = React.lazy(...)
const CopilotMobile = React.lazy(...)
const StudioMobile = React.lazy(...)
```

**响应式切换：**
```typescript
const isMobile = useIsMobile();

if (isMobile) {
  return <MobileVersion />;
}
return <DesktopVersion />;
```

**减少bundle体积：** 移动端不加载桌面端代码

---
